# clsString

A custom C++ string utility class built from scratch for learning **Object-Oriented Programming (OOP)** and understanding how common string operations work internally.

## Features

### Reading
- Print vowels
- Print words
- Print first letter of each word

### String Manipulation
- Trim left/right spaces
- Normalize spaces
- Convert to upper/lower case
- Invert character case
- Remove punctuation

### Processing
- Split strings
- Join strings
- Replace words

### Statistics
- Count words
- Count vowels
- Count uppercase letters
- Count lowercase letters
- Count character occurrences

## Example

```cpp
clsString Text("Hello, World!");

cout << Text.CountWords() << endl;
cout << Text.ConvertToUpperCase() << endl;
cout << Text.CountVowels() << endl;
```
